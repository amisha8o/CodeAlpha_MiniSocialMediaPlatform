# CodeAlpha_MiniSocialMediaPlatform

A modern and responsive *Mini Social Media Platform* developed using the *MERN Stack* during the *CodeAlpha Full Stack Development Internship*.  
This platform allows users to create posts, interact with others, like posts, comment, and experience core social media functionalities through a clean and interactive interface.

---

## 🚀 Features

✅ User Authentication System  
✅ Create, Edit & Delete Posts  
✅ Like & Comment Functionality  
✅ User Profile Section  
✅ Responsive User Interface  
✅ REST API Integration  
✅ MongoDB Database Connectivity  
✅ Modern & Interactive Design
